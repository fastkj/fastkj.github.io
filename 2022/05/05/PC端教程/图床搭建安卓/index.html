GitHub ＋ PicGo 图床搭建教程 无限存储空间

因为需要备份大量的图片，但市面上很多免费的图床都有限制，国内第三方图床配置应用都需要会员，所以比较推荐 GitHub ＋ PicGo 配置图床,平时用的都是 PC 端 PicGo，最近发现 PicGo 有安卓版，写一个配置教程。

优点：无限存储空间、秒上传下载、自定义图床链接

开源项目就不搬运了自行到仓库下载

Flutter-PicGo 仓库：https://github.com/PicGo/flutter-picgo/releases



配置方式

  1. 设定分支名 (必填)：
    仓库分支名填写”master” 或者”main”

  2. 自定义域名(可填镜像源提升上传速度)：
    镜像源：https://fastly.jsdelivr.net/gh/仓库名

  3. 指定存储路径：
    这个选项可以为空，如果想将图片上传到仓库的指定目录下，可以填写 img/

  4. 按照 “GitHub 账户名 / 仓库名的格式填写”，比如我的是：Fast/picture

  5. 设定 Token (必填)：
    将之前步骤的 Token 值复制粘贴到这里
