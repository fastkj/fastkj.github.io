Termux 基于安卓 Android 手机的一款高级终端

Termux - 高级终端

Termux 是一个 Android 下一个高级的终端模拟器, 开源且不需要 root，支持 apt 管理软件包，十分方便安装软件包, 完美支持 Python, PHP, Ruby, Go, Nodejs, MySQL 等。

随着智能设备的普及和性能的不断提升，如今的手机、平板等的硬件标准已达到了初级桌面计算机的硬件标准, 用心去打造完全可以把手机变成一个强大的工具。


安装 Hexo

个人用 Termux 主要写 Hexo 因为一共就那几条命令 打开电脑太麻烦了，当然你也可以搭建完整的 Linux 环境，看个人需求。

Hexo 安装命令：

termux-setup-storage

pkg install nodejs-lts vim git -y

pkg update && pkg upgrade

npm install -g hexo-cli

pkg install nodejs
